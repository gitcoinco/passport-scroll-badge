# Passport Score Badge

A Badge for Scroll's Canvas application.

## Building

Install foundryup and forge using instructions [here](https://book.getfoundry.sh/getting-started/installation)
then run:

```bash
yarn
yarn build
```

## Testing

```bash
yarn test
```

## Deploy to Scroll Mainnet

```bash
yarn deploy
```
